You can use this screen to report an error by email to the author of .

When you select the button below, an email will be generated that contains:

  • The last Java exceptions (a.k.a. 'force close').
  • The Android phone model.
  • The generic actions logged by the application.
No personal identification is reported.
The mail will open in your default mail application where you get a chance to review and edit it before sending it.
